Jumpdude 
2d platformer

Builds on UE4.2.7

Controls
-------------
Move with arrows  
Jump or Double Jump with Space  
Dash with Shift or F  

Using free assets from: https://www.unrealengine.com/marketplace/en-US/product/kenney-nl-platformer | https://kenney.nl/
